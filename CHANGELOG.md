## 使用说明

1. **下载并打开 DMG**
   - 双击 `OpenClawInstaller.dmg`
   - 将应用拖拽到 Applications 文件夹

2. **启动安装程序**
   - 从 Applications 运行 OpenClawInstaller
   - 授予管理员权限（首次运行）

3. **自动化安装流程**
   - 系统环境检测
   - 自动选择最优下载源
   - Node.js 安装（如需要）
   - OpenClaw 安装
   - 配置向导
