# OpenClaw Installer - 更新日志

## v1.1.12 (2026-03-14)

### ✨ Chat 新功能

- **附加文件** — 输入框新增回形针按钮，支持选择或拖拽附加图片、PDF、Office 文档、音视频等文件，发送时一并提交给 AI
- **@ 提及切换 Agent** — 输入框中输入 `@` 弹出 Agent 列表，选择后直接发消息给指定 Agent，无需先切换
- **Skills 面板** — 输入 `/skills` 弹出技能列表，支持搜索筛选和键盘导航，快速调用已安装技能
- **输入历史回溯** — 输入框为空时按 ↑/↓ 键可回看最近 20 条已发送消息，重启后保留

### 🔄 后台任务机制

- **转后台按钮** — 任务运行超过 60 秒后出现"转后台"按钮，点击后解锁输入框继续聊天；超过 120 秒自动转入后台
- **完成通知与跳转** — 后台任务完成后在聊天底部显示 ✅ 通知，点击"查看结果 ↑"跳转至对应回复
- **超时处理修正** — 任务超时上限延长至 30 分钟；超时后准确显示 ⚠️ 超时提示，不再错误显示为"已完成"
- **Streaming 竞态修复** — 修复后台任务完成后结果被中间输出覆盖导致内容丢失的问题

### 🌍 国际化

- **深度 i18n 修复** — 修复 `String(localized:)` 不跟随应用内语言切换的根本问题：View 层改用 SwiftUI 原生 `Text(LocalizedStringKey)`，ViewModel 层统一使用 `LanguageManager.localizedBundle`
- **新增翻译** — 所有新功能的用户可见文本均已完成 25 种语言翻译
- **清理旧 key** — 移除不再使用的国际化条目

### 🐛 Bug 修复

- **切换 Tab 滚动** — 修复从其他页面切回 Chat 时消息停留在顶部、未自动滚到最新消息的问题（macOS 14+ 使用 `defaultScrollAnchor(.bottom)`，macOS 13 降级兼容）
- **参数化字符串 key 不匹配** — 修复 `String(localized:)` 使用 `\()` 插值导致与 xcstrings 中 `%@` 格式 key 不匹配的翻译失效问题

### 📖 文档

- **用户指南更新** — 新增附加文件、@ 提及、Skills 面板、后台任务与超时机制的使用说明；斜杠命令表格补充 `/skills`

## 使用说明

1. **下载并打开 DMG**
   - 双击 `OpenClawInstaller.dmg`
   - 将应用拖拽到 Applications 文件夹

2. **启动安装程序**
   - 从 Applications 运行 OpenClawInstaller
   - 授予管理员权限（首次运行）

3. **自动化安装流程**
   - 系统环境检测
   - 自动选择最优下载源
   - Node.js 安装（如需要）
   - OpenClaw 安装
   - 配置向导
